# 社区业务应用现状

## 三维识别框架
- **X轴**：患者疾病进展周期  
  早筛预防 → 多线治疗过程 → 关键支持治疗 → 全程自我管理（病情/心理/营养/疼痛）→ 家属PTSD支持

- **Y轴**：信息差分类  
  公益应用为主，避开商业应用聚集点，针对患者群体特定需求

- **Z轴**：病情种类优先级  
  优先满足病情危重、复杂度高、预后差的病种应用
